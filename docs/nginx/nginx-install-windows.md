手把手带你在 Windows 上安装 Nginx-1.16.0 。
<!-- more -->

```bash
# 下载免安装版本的 Nginx
# http://nginx.org/en/download.html
# nginx-1.16.1.zip

# 解压
D:\nginx-1.16.0

# 修改配置文件（如果需要）
D:\nginx-1.16.0\conf\nginx.conf

# 启动服务（如果需要，可加入系统服务）
D:\nginx-1.16.0\nginx.exe
```
